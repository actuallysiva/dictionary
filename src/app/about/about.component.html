Oxford Dictionary API

The Oxford Dictionaries API allows easy access to our world-renowned dictionary content. Use the live documentation below to try it out, view real responses, and explore a growing number of code samples to help you get started. As a registered user, use https://od-api.oxforddictionaries.com/api/v1 as base URL for API requests. Take a look at our FAQ page if you have any questions, or feel free to contact us.
